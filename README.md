# RN2483-LoRa-Shield
RN2484 LoRa Shield for Arduino UNO, currently available via <http://www.ebay.co.uk/itm/Wireless-868-MHz-LoRa-Shield-for-Arduino-UNO-Microchip-RN2483-/292014187013>

PIN Layout:
 * Reset -- 4
 * Uart TX -- 7
 * Uart RX -- 8
